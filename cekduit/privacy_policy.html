Kebijakan Privasi CekDuit

Terakhir diperbarui: 15 Januari 2026

Selamat datang di CekDuit! Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi CekDuit.

1. Informasi yang Kami Kumpulkan

1.1 Informasi yang Anda Berikan

  • Data Akun: Email dan kata sandi (jika Anda memilih untuk membuat akun)
  • Data Keuangan: Transaksi pemasukan dan pengeluaran yang Anda input secara manual
  • Kategori Kustom: Kategori yang Anda buat untuk mengorganisir transaksi

1.2 Informasi yang Dikumpulkan Secara Otomatis

  • Data Perangkat: Jenis perangkat, versi sistem operasi, dan pengidentifikasi perangkat unik
  • Data Penggunaan: Fitur yang digunakan, waktu penggunaan aplikasi
  • Data Crash: Laporan error untuk meningkatkan stabilitas aplikasi

1.3 Informasi dari Pihak Ketiga

  • Google Play: Informasi pembelian untuk langganan premium
  • Google AdMob: Data iklan untuk pengguna versi gratis

2. Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan informasi yang dikumpulkan untuk:

  • Menyediakan layanan pencatatan keuangan
  • Menyinkronkan data antar perangkat (pengguna Premium)
  • Memproses pembayaran langganan
  • Menampilkan iklan yang relevan (pengguna Gratis)
  • Mengirim notifikasi pengingat (jika diaktifkan)
  • Meningkatkan dan mengembangkan fitur aplikasi
  • Mendeteksi dan mencegah penipuan

3. Penyimpanan Data

3.1 Penyimpanan Lokal

Data transaksi disimpan secara lokal di perangkat Anda menggunakan SQLite. Data lokal sepenuhnya berada di bawah kendali Anda.

3.2 Penyimpanan Cloud (Pengguna Premium)

Data disinkronkan ke server Supabase yang aman. Server berlokasi di infrastruktur yang sesuai dengan standar keamanan industri. Data dienkripsi saat transit menggunakan TLS/SSL.

4. Berbagi Informasi

Kami TIDAK menjual data pribadi Anda. Kami hanya berbagi informasi dengan pihak ketiga yang diperlukan untuk menyediakan layanan.
Pihak Ketiga Tujuan Data yang Dibagikan
Google Play Pemrosesan pembayaran Info transaksi pembelian
Google AdMob Menampilkan iklan ID perangkat, data iklan
Supabase Penyimpanan cloud Data transaksi (Premium)
Firebase Analitik & crash report Data penggunaan anonim

5. Keamanan Data

Kami menerapkan langkah-langkah keamanan berikut:

  • Enkripsi data saat transit (TLS/SSL)
  • Autentikasi aman untuk akun pengguna
  • Akses terbatas ke data pengguna
  • Pemantauan keamanan berkala
  • Kata sandi di-hash dan tidak disimpan dalam bentuk teks biasa

6. Hak Privasi Anda

6.1 Mengakses Data

Anda dapat melihat semua data yang tersimpan di aplikasi kapan saja.

6.2 Menghapus Data

  • Data Lokal: Hapus aplikasi untuk menghapus semua data lokal
  • Data Cloud: Hubungi kami di cekduit.app@gmail.com untuk menghapus data cloud

6.3 Mengekspor Data

Fitur ekspor laporan tersedia untuk mengunduh data transaksi Anda.

6.4 Menolak Iklan Personalisasi

Anda dapat menonaktifkan iklan personalisasi melalui pengaturan perangkat Android.

7. Penggunaan oleh Anak-anak

CekDuit tidak ditujukan untuk anak-anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan informasi dari anak-anak di bawah 13 tahun.

8. Iklan

8.1 Pengguna Gratis

Aplikasi menampilkan iklan dari Google AdMob. Iklan ini mungkin menggunakan ID perangkat, data lokasi umum, dan riwayat interaksi iklan.

8.2 Pengguna Premium

Pengguna Premium menikmati pengalaman bebas iklan.

9. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diberitahukan melalui pembaruan tanggal di halaman ini dan notifikasi dalam aplikasi untuk perubahan signifikan.

Data Transaksi (Cloud) Sampai akun dihapus atau diminta penghapusan
